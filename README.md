# AutoGain-SoundSage
**WORK IN PROGRESS **
Open source Python program for automating gain staging using natural language processing. part 1 of a series for automating audio processing tasks, end goal is to create a full set of tools for an AI to use for automating Audio processing for Music, Film, Game and any other possible applications. UI is very basic but 'AutoGain' the original gain-staging application is very functional.
